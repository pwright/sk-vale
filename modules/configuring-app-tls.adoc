== Application TLS configuration

[role="_abstract"]
To configure TLS certificates for the {registry} application, provide keystore references in the `ApicurioRegistry3` custom resource.

[source,yaml]
----
spec:
  app:
    tls:
      keystoreSecretRef:
        name: registry-tls
        key: keystore.p12
      keystorePasswordSecretRef:
        name: registry-tls
        key: keystore.password
----
