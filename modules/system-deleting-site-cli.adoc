== Deleting a site using the CLI on local systems

:_mod-docs-content-type: REFERENCE

[role="_abstract"]
Use the CLI to remove an existing Skupper site from a local system.

*Prerequisites*

* The `skupper` CLI is installed.

*Procedure*

. Enter the following command to delete a site:
+
[,bash]
----
skupper site delete <sitename>
skupper system stop
----

. You can also uninstall the controller after deleting all existing sites:
+
[,bash]
----
skupper system uninstall
----
