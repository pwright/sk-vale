= configuring-networkpolicy_{context}

== NetworkPolicy configuration

[role="_abstract"]
The operator creates NetworkPolicy resources by default. To manage NetworkPolicy manually, disable the operator-managed NetworkPolicy.

[source,yaml]
----
spec:
  app:
    networkPolicy:
      enabled: false
  ui:
    networkPolicy:
      enabled: false
----
